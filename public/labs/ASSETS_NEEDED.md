# Labs Page Assets

This directory contains all assets for the `/labs` page.

## Required Assets

### Videos (`/public/labs/videos/`)
- `START_LABS_trimmed_video.mp4` - Hero section background video
- `START_LABS_poster.jpg` - Video poster image (fallback for when video doesn't load)

### Images (`/public/labs/images/`)

#### Logos
- `startmunich.png` - START Munich logo (used in hero and partners section)
- `oneaim.png` - OneAIM logo (used in hero and partners section)

#### Background
- `rocket.png` - Rocket background image for About section

#### Proof Section Carousel
- `letter.JPG` - Carousel image 1
- `demoday.jpg` - Carousel image 2
- `award.png` - Carousel image 3
- `winner.jpg` - Carousel image 4
- `partner.jpg` - Carousel image 5

## Notes

- Ensure video file is optimized for web (H.264 codec recommended)
- Poster image should be a representative frame from the video
- All images should be optimized for web delivery
- Consider using Next.js Image optimization for static images
